Detailed Description

Implementation of a table-driven lexer.

This class performs tokenisation of the program string by looking up the appropriate DFA transitions in a transition table.

Constructor & Destructor Documentation

Lexer::Lexer ( std::string &  program)

Constructor for Lexer class with the input program string as a parameter.

This constructor generates the tokens by successively executing the function next_token(std::string&, unsigned int&), until the end of the string is encountered. This process populates the global vector tokens.
